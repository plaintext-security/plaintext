# Track 01 — Offensive Security

Learn to think like an attacker. Reconnaissance, exploitation,
post-exploitation, and professional reporting.

## Modules (coming soon)
- 01 Reconnaissance (nmap, theHarvester)
- 02 Exploitation Fundamentals (metasploit)
- 03 Web Application Attacks (burpsuite CE, sqlmap)
- 04 Post-Exploitation (linpeas, pwncat)
- 05 Reporting

## Prerequisites
Complete Track 00 — Foundations first.

> All labs use intentionally vulnerable targets (DVWA, HackTheBox free tier,
> TryHackMe free rooms, or locally spun VMs). Never test on systems you
> don't own or have explicit written permission to test.

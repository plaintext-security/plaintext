# plaintext

> Open security education for everyone. No paywalls. No gatekeeping. Forever free.

Plaintext is a community-built cybersecurity curriculum using only open source tools.
Professional-grade training from foundations to advanced specialisations — free to use,
free to fork, free to improve.

## Tracks

| # | Track | Tools |
|---|-------|-------|
| 00 | Foundations | tcpdump, openssl, python, bash |
| 01 | Offensive Security | nmap, metasploit, burpsuite CE, sqlmap |
| 02 | Defensive Operations | elastic, zeek, suricata, wazuh |
| 03 | Digital Forensics | autopsy, volatility, wireshark, sleuthkit |
| 04 | Malware Analysis | ghidra, cutter, cuckoo, remnux |
| 05 | Cloud Security | prowler, pacu, cloudsploit, trivy |

## Structure

```
tracks/
└── 00-foundations/
    └── modules/
        └── 01-networking/
            ├── README.md   ← concept explanation
            └── lab.md      ← hands-on exercise
```
